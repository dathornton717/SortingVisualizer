# SortingVisualizer

Visually sorts a list of numbers with the following sorting algorithms:
* BogoBogo Sort
* Bogo Sort
* Bubble Sort
* Cocktail Shaker Sort
* Gnome Sort
* Heap Sort
* Insertion Sort
* Quick Sort
* Radix Sort
* Selection Sort
* Shell Sort
* Stooge Sort

To run:
Run the main method in Main.java
